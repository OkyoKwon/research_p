# 서비스 딥리서치

특정 서비스나 제품에 대해 다각도의 심층 리서치를 수행하는 하네스입니다. 시장 현황, 경쟁사 분석, 사용자 리뷰, 기술 스택, 비즈니스 모델 등 다양한 관점에서 정보를 수집하고 종합합니다. 최종적으로 구조화된 리서치 보고서를 생성하여 의사결정에 필요한 인사이트를 제공합니다.

## 에이전트 구성

- **market-analyst**: 서비스 시장 현황 및 산업 트렌드 분석가
- **competitor-analyst**: 경쟁사 심층 분석가
- **user-review-analyst**: 사용자 리뷰 및 소비자 인사이트 분석가
- **biz-tech-analyst**: 비즈니스 모델 및 기술 스택 분석가
- **research-reviewer**: 리서치 통합 검증 및 최종 보고서 작성자

# 유저/서비스 시나리오 작성 및 HTML 시각화

사용자 또는 서비스의 요구사항을 입력받아 구체적인 사용 시나리오를 자동으로 작성합니다. 작성된 시나리오를 기반으로 흐름도, 다이어그램, 스토리보드 등을 HTML 형태로 시각화하여 직관적으로 확인할 수 있게 제공합니다. 기획자, 디자이너, 개발자가 시나리오를 빠르게 공유하고 검토할 수 있도록 완성된 HTML 파일을 최종 산출물로 출력합니다.

## 에이전트 구성

- **scenario-analyst**: 요구사항을 분석하여 구조화된 사용 시나리오를 작성
- **flow-designer**: 시나리오를 기반으로 흐름도와 다이어그램 데이터를 설계
- **frontend-dev**: 프론트엔드 개발자
- **qa-engineer**: 시나리오 완결성 검토 및 HTML 파일 품질 검증

# 내부 공유용 PPTX 작성

사용자가 제공한 주제나 내용을 바탕으로 내부 공유용 파워포인트 발표 자료를 자동으로 작성합니다. 슬라이드 구성, 핵심 메시지 정리, 시각적 레이아웃 제안까지 포함한 PPTX 파일을 생성합니다. 팀 내 보고, 업무 공유, 프로젝트 현황 전달 등 다양한 내부 커뮤니케이션 목적에 맞게 활용할 수 있습니다.

## 에이전트 구성

- **content-planner**: 발표 목적과 청중을 분석하고 슬라이드 구성안(목차)을 설계한다
- **slide-writer**: 슬라이드 구성안을 바탕으로 각 슬라이드의 본문 콘텐츠를 작성한다
- **visual-designer**: 슬라이드별 레이아웃, 색상 테마, 시각화 요소를 설계하고 Mermaid 다이어그램을 작성한다
- **pptx-builder**: 모든 산출물을 종합하여 python-pptx로 실제 PPTX 파일을 생성한다

# 서비스 심층 리서치

특정 서비스나 제품에 대해 다각도로 심층 조사를 수행하는 하네스입니다. 시장 현황, 경쟁사 분석, 사용자 리뷰, 기술 스택, 비즈니스 모델 등 다양한 관점에서 정보를 수집하고 분석합니다. 최종적으로 구조화된 리서치 보고서를 생성하여 의사결정에 필요한 인사이트를 제공합니다.

## 에이전트 구성

- **industry-analyst**: 서비스 대상 산업의 시장 현황과 구조를 분석한다
- **competitor-analyst**: 경쟁 서비스를 다각도로 비교 분석한다
- **consumer-analyst**: 서비스 사용자의 리뷰와 니즈를 분석한다
- **business-modeler**: 서비스의 비즈니스 모델과 수익 구조를 분석한다
- **research-reviewer**: 전체 리서치 결과를 검증하고 최종 통합 보고서를 생성한다
