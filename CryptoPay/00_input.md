# 크립토 결제서비스 Flow 정리 및 시각화 요청

## 요청 내용
현재 workspace에 작성된 4개 크립토 결제서비스들의 Flow를 정리 및 시각화

## 대상 서비스
1. **Base Pay** - Coinbase L2 블록체인 Base 네트워크 기반 결제 프로토콜
2. **Binance Pay** - Binance 거래소 기반 오프체인 암호화폐 결제 서비스
3. **Coinbase Commerce** - Coinbase의 암호화폐 결제 게이트웨이
4. **Stripe Crypto** - Stripe의 스테이블코인/크립토 결제 서비스

## 시각화 범위
각 서비스별 주요 시나리오:
- **결제(Payment)** 시나리오 (즉시결제, 에스크로, P2P, QR, 머신결제 등)
- **정산(Settlement)** 시나리오 (온체인 즉시, 오프체인, 법정화폐 전환 등)
- **환불(Refund)** 시나리오 (전액환불, 부분환불, Void, Reclaim 등)
