# BitPay 서비스 심층 리서치 요청

## 분석 대상
- **서비스명**: BitPay (https://bitpay.com)
- **서비스 유형**: 암호화폐 결제 게이트웨이 / 결제 프로세서
- **분석 일자**: 2026-04-16

## 분석 요구사항
1. BitPay 서비스 전반에 대한 심층 분석
2. **결제(Payment) 시나리오 상세** - 모든 결제 방식의 step-by-step 흐름
3. **정산(Settlement) 시나리오 상세** - 법정화폐/암호화폐 정산 흐름, 속도, 수수료
4. **환불(Refund) 시나리오 상세** - 환불 프로세스, 정책, 분쟁 해결

## 분석 범위
- 서비스 개요 및 핵심 수치
- 결제/정산/환불 시나리오 상세
- 시장 현황 및 경쟁 환경 (기존 리서치된 Base Pay, Binance Pay, Coinbase Commerce, Stripe Crypto와의 비교 포함)
- 비즈니스 모델 및 기술 아키텍처
- 사용자 인사이트 (리뷰, Pain Points, 만족 요소)
- 전략적 시사점

## 기존 리서치 컨텍스트
동일 프로젝트에서 이미 분석 완료된 서비스:
- Base Pay (`CryptoPay/service_research/Base Pay/`)
- Binance Pay (`CryptoPay/service_research/Binance Pay/`)
- Coinbase Commerce (`CryptoPay/service_research/Coinbase commerce/`)
- Stripe Crypto (`CryptoPay/service_research/Stripe Crypto/`)

BitPay는 이들 보고서에서 주요 경쟁사로 자주 언급되었으며, 시장점유율 1위(20%)로 평가됨.
